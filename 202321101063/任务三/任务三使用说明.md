## 三、界面布局

```
┌──────────────────────────────────────┐
│  输入框                               │  ← 键盘输入 / 按钮点击
│                           = 结果      │  ← 绿色结果区
│                          错误提示     │  ← 红色错误区
│  计算记录 (最多 20 条)    [清空]      │
```

## 四、支持的运算

### 基本运算

| 运算 | 符号 | 示例 |
|------|------|------|
| 加减乘除 | `+ - * /` | `1 + 2 * 3` |
| 括号 | `( )` | `(2 + 3) * 4` |
| 负数 | `-` 前缀 | `-5 + 3` |
| 乘方 | `^` | `2^3` |

### 函数

| 函数 | 示例 | 结果 |
|------|------|------|
| `pow(m, n)` | `pow(8, 1/3)` | `2`（开立方） |
| `sqrt(x)` | `sqrt(25)` | `5` |
| `sin/cos/tan(x)` | `sin(0)` | `0` |
| `log/log10/log2(x)` | `log10(100)` | `2` |
| `exp(x)` | `exp(1)` | `2.718...` |
| `abs(x)` | `abs(-5)` | `5` |

---

## 五、计算记录（新增）

- 每次成功计算自动保存，**最多 20 条**，超出自动删除最早记录
- **点击任意记录** → 表达式回填到输入框，可修改后重新计算
- 点击 **「清空」** 按钮删除全部记录

---

## 六、快捷键

| 按键 | 功能 |
|------|------|
| `Enter` | 计算 |
| `Esc` | 清空输入框 |

---

## 七、错误提示

| 提示 | 原因 |
|------|------|
| `语法错误` | 括号不配对或运算符位置不对 |
| `数学错误: 除以零` | 分母为 0 |
| `无法识别的字符` | 输入了不支持的特殊字符 |
| `未知函数` | 函数名拼写错误 |
