# Vue 网页科学计算器 — 使用守则

## 概述

本程序是一个基于 Vue 3 + Vite 构建的响应式网页科学计算器，采用递归下降解析引擎，支持实数域内的科学计算。适配桌面端、平板和手机端，暗色主题设计。编译产物为静态文件 `dist/`，可直接部署网页或通过 HBuilder 打包为 Android APK。

## 使用方式

### 网页端直接打开

浏览器直接打开 `calculator-vue/dist/index.html` 即可使用。

### 本地开发调试

```bash
cd calculator-vue
npm install
npm run dev
```

启动 Vite 开发服务器，浏览器访问 `http://localhost:5173`。

### 构建生产版本

```bash
cd calculator-vue
npm run build
```

生成 `dist/` 目录，包含 `index.html` + CSS + JS，所有资源使用相对路径（`base: './'`），可直接部署到任意静态服务器。

### 打包 Android APK

1. 执行 `npm run build` 生成 `dist/`
2. 将 `dist/` 整个目录导入 HBuilder
3. 在 HBuilder 中选择「云打包」→ 生成 APK

> 注意：`vite.config.js` 中已配置 `base: './'`，确保 APK 内相对路径正常工作。

## 输入方式

### 按钮键盘输入

点击界面按钮即可追加字符到表达式输入框：

| 按钮类别 | 按钮内容 | 说明 |
|----------|----------|------|
| 功能键 | `C` `CE` `⌫` | C=清除输入，CE=全部清除，⌫=退格 |
| 科学函数 | `sin` `cos` `tan` `ln` `log` `√` `abs` `pow` | 点击自动插入函数名+左括号 |
| 常量 | `π` | 插入常量 `pi` |
| 运算符 | `÷` `×` `−` `+` | 对应 / * - + |
| 数字 | `0`-`9` `.` | 数字和小数点 |
| 括号 | `(` `)` | 左右括号 |
| 乘方 | `^` | 乘方运算符 |
| 等号 | `=` | 计算结果 |

### 键盘输入

| 键盘按键 | 功能 |
|----------|------|
| `0`-`9` `.` | 数字和小数点 |
| `+` `-` `*` `/` | 四则运算符 |
| `^` | 乘方 |
| `(` `)` `,` | 括号和逗号 |
| `Enter` | 计算结果 |
| `Backspace` | 退格删除 |
| `Escape` | 清除输入 |

## 支持的运算

### 四则运算

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `+` | 加法 | `3 + 5 → 8` |
| `-` | 减法 | `9 - 2 → 7` |
| `*` | 乘法 | `4 * 6 → 24` |
| `/` | 除法 | `8 / 2 → 4` |

### 乘方运算

| 语法 | 含义 | 示例 |
|------|------|------|
| `m ^ n` | m 的 n 次方 | `2 ^ 3 → 8` |
| `pow(m, n)` | m 的 n 次方 | `pow(2, 3) → 8` |

- 乘方为**右结合**：`2 ^ 3 ^ 2 = 512`
- `pow` 参数可留空，默认值为 1

### 开方运算

| 语法 | 含义 | 示例 |
|------|------|------|
| `sqrt(x)` | 平方根 | `sqrt(9) → 3` |
| `pow(x, 1/n)` | n 次方根 | `pow(8, 1/3) → 2` |
| `x ^ (1/n)` | n 次方根 | `9 ^ (1/2) → 3` |

### 三角函数

| 函数 | 含义 | 示例 |
|------|------|------|
| `sin(x)` | 正弦（弧度） | `sin(pi/6) → 0.5` |
| `cos(x)` | 余弦（弧度） | `cos(0) → 1` |
| `tan(x)` | 正切（弧度） | `tan(pi/4) → 1` |

### 对数函数

| 函数 | 含义 | 示例 |
|------|------|------|
| `log(x)` | 常用对数（log₁₀） | `log(100) → 2` |
| `ln(x)` | 自然对数（logₑ） | `ln(e) → 1` |

### 其他函数与常量

| 名称 | 含义 | 示例 |
|------|------|------|
| `abs(x)` | 绝对值 | `abs(-5) → 5` |
| `pi` | 圆周率 π ≈ 3.14159 | 点击 `π` 按钮 |
| `e` | 自然常数 ≈ 2.71828 | 键盘输入 `e` |

## 括号与优先级

优先级从高到低：

1. 括号 `( )`
2. 乘方 `^` / 函数调用
3. 一元正负号 `+` / `-`
4. 乘除 `*` / `/`
5. 加减 `+` / `-`

## 响应式适配

| 屏幕宽度 | 设备类型 | 适配策略 |
|----------|----------|----------|
| ≥ 768px | 桌面端 | 正常布局，hover 效果生效 |
| ≤ 480px | 手机端 | 字体缩小、间距收紧、圆角减小 |
| ≤ 360px | 小屏手机 | 进一步压缩字体和间距 |

> 按钮在移动端使用 `touch` 优化，`:active` 状态替代 `:hover`，避免 300ms 点击延迟。

## 错误处理

| 错误类型 | 触发条件 | 示例 |
|----------|----------|------|
| 除以零 | 分母为零 | `1 / (2 - 2)` |
| 实数域错误 | 负底数的非整数次方 | `(-1) ^ 0.5` |
| sqrt 域错误 | sqrt 参数为负数 | `sqrt(-4)` |
| 语法错误 | 表达式格式不正确 | `2 + * 3` |
| 未识别名称 | 未注册的函数/常量名 | `foo(1)` |

## 项目结构

```
calculator-vue/
├── package.json              # Vue 3.5 + Vite 6
├── vite.config.js            # base: './'（相对路径，APK 兼容）
├── index.html                # 入口 HTML
├── dist/                     # ✅ 构建产物
│   ├── index.html
│   └── assets/
│       ├── index-*.css
│       └── index-*.js
└── src/
    ├── main.js               # Vue 应用入口
    ├── App.vue               # 主组件（状态管理 + 键盘监听）
    ├── engine/
    │   └── calculatorEngine.js  # 计算引擎（Lexer + Parser + evaluate）
    └── components/
        ├── CalcDisplay.vue   # 显示区组件
        └── CalcButtons.vue   # 按钮网格组件
```

## 技术栈

- **Vue 3.5** — Composition API + `<script setup>`
- **Vite 6** — 构建工具
- **纯 JS 计算引擎** — 无第三方依赖，递归下降解析器
- **CSS 响应式** — 无 UI 框架依赖，纯手写样式
