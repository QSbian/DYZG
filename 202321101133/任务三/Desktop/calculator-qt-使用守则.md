# Qt C++ 科学计算器 — 使用守则

## 概述

本程序是一个基于 Qt 6.11.0 + MinGW 13.1 编写的桌面 GUI 科学计算器，采用递归下降解析引擎，支持实数域内的科学计算，提供表达式输入框和按钮键盘双输入方式。

## 启动方式

双击 `QtCalculator/bin/Calculator.exe` 直接运行。

> 程序已通过 `windeployqt` 打包所有 Qt 依赖 DLL，无需额外安装 Qt 环境即可运行。

## 输入方式

### 按钮键盘输入

点击界面上的按钮即可向表达式输入框追加字符：

| 按钮类别 | 按钮内容 | 说明 |
|----------|----------|------|
| 功能键 | `C` `CE` `⌫` | C=清除输入，CE=全部清除，⌫=退格删除 |
| 科学函数 | `sin` `cos` `tan` `ln` `log` `√` `abs` `pow` | 点击后自动插入函数名和左括号 |
| 常量 | `π` | 插入常量 `pi` |
| 运算符 | `÷` `×` `−` `+` `^` | 对应 / * - + ^ |
| 数字 | `0`-`9` `.` | 数字和小数点 |
| 括号 | `(` `)` | 左右括号 |
| 等号 | `=` | 计算结果 |

### 键盘输入

表达式输入框支持直接键盘打字，键盘映射：

| 键盘按键 | 功能 |
|----------|------|
| `0`-`9` `.` | 数字和小数点 |
| `+` `-` `*` `/` | 四则运算符 |
| `^` | 乘方 |
| `(` `)` | 括号 |
| `Enter` | 计算结果 |
| `Backspace` | 退格删除 |
| `Escape` | 清除输入 |

> 注意：函数名（sin、cos 等）和常量名（pi、e）只能通过按钮或键盘手动拼写输入。

## 支持的运算

### 四则运算

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `+` | 加法 | `3 + 5 → 8` |
| `-` | 减法 | `9 - 2 → 7` |
| `*` | 乘法 | `4 * 6 → 24` |
| `/` | 除法 | `8 / 2 → 4` |

### 乘方运算

| 语法 | 含义 | 示例 |
|------|------|------|
| `m ^ n` | m 的 n 次方 | `2 ^ 3 → 8` |
| `pow(m, n)` | m 的 n 次方 | `pow(2, 3) → 8` |

- 乘方为**右结合**：`2 ^ 3 ^ 2 = 512`
- `pow` 参数可留空，默认值为 1

### 开方运算

| 语法 | 含义 | 示例 |
|------|------|------|
| `sqrt(x)` | 平方根 | `sqrt(9) → 3` |
| `pow(x, 1/n)` | n 次方根 | `pow(8, 1/3) → 2` |
| `x ^ (1/n)` | n 次方根 | `9 ^ (1/2) → 3` |

### 三角函数

| 函数 | 含义 | 示例 |
|------|------|------|
| `sin(x)` | 正弦（弧度） | `sin(pi/6) → 0.5` |
| `cos(x)` | 余弦（弧度） | `cos(0) → 1` |
| `tan(x)` | 正切（弧度） | `tan(pi/4) → 1` |

### 对数函数

| 函数 | 含义 | 示例 |
|------|------|------|
| `log(x)` | 常用对数（log₁₀） | `log(100) → 2` |
| `ln(x)` | 自然对数（logₑ） | `ln(e) → 1` |

### 其他函数与常量

| 名称 | 含义 | 示例 |
|------|------|------|
| `abs(x)` | 绝对值 | `abs(-5) → 5` |
| `pi` | 圆周率 π ≈ 3.14159 | 点击 `π` 按钮 |
| `e` | 自然常数 ≈ 2.71828 | 键盘输入 `e` |

## 括号与优先级

优先级从高到低：

1. 括号 `( )`
2. 乘方 `^` / 函数调用
3. 一元正负号 `+` / `-`
4. 乘除 `*` / `/`
5. 加减 `+` / `-`

## 结果显示

- 计算成功：结果显示为绿色
- 计算错误：错误信息显示为红色
- 历史记录区保留最近的计算记录

## 错误处理

| 错误类型 | 触发条件 | 示例 |
|----------|----------|------|
| 除以零 | 分母为零 | `1 / (2 - 2)` |
| 实数域错误 | 负底数的非整数次方 | `(-1) ^ 0.5` |
| sqrt 域错误 | sqrt 参数为负数 | `sqrt(-4)` |
| 语法错误 | 表达式格式不正确 | `2 + * 3` |

## 重新编译

如需修改代码后重新编译：

```bat
cd QtCalculator
PATH=D:\QT\Tools\mingw1310_64\bin;D:\QT\6.11.0\mingw_64\bin;%PATH%
qmake Calculator.pro
mingw32-make -f Makefile.Release -j4
```

或直接运行 `build.bat`。

## 用 Qt Creator 打开

双击 `Calculator.pro`，Qt Creator 会自动识别项目配置，可在 IDE 中编辑代码、调试和运行。

> Qt Creator 安装路径：`D:\QT\Tools\QtCreator\`
