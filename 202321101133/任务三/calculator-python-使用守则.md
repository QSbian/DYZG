# Python 命令行科学计算器 — 使用守则

## 概述

本程序是一个 Python 编写的交互式命令行科学计算器，基于递归下降解析器实现，支持实数域内的科学计算。

## 启动方式

### 交互模式（推荐）

```bash
python calculator.py
```

进入 REPL 循环，持续输入表达式、即时输出结果。输入 `quit` / `exit` / `q` 退出。

### 单次计算模式

```bash
python calculator.py "表达式"
```

计算一个表达式后直接退出，适合脚本调用。

> 示例：`python calculator.py "1+5*pow(,1/2)/(2*(-5.2)^3)"`

## 支持的运算

### 四则运算

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `+` | 加法 | `3 + 5 → 8` |
| `-` | 减法 | `9 - 2 → 7` |
| `*` | 乘法 | `4 * 6 → 24` |
| `/` | 除法 | `8 / 2 → 4` |

运算优先级：乘除 > 加减，与数学规则一致。

### 乘方运算

| 语法 | 含义 | 示例 |
|------|------|------|
| `m ^ n` | m 的 n 次方 | `2 ^ 3 → 8` |
| `pow(m, n)` | m 的 n 次方 | `pow(2, 3) → 8` |

- 乘方为**右结合**：`2 ^ 3 ^ 2 = 2 ^ (3 ^ 2) = 512`
- 乘方优先级高于乘除：`2 * 3 ^ 2 = 2 * 9 = 18`
- `pow` 参数可以留空，默认值为 1：`pow(, 1/2) = pow(1, 1/2) = 1`

### 开方运算

| 语法 | 含义 | 示例 |
|------|------|------|
| `sqrt(x)` | 平方根 | `sqrt(9) → 3` |
| `pow(x, 1/n)` | n 次方根 | `pow(8, 1/3) → 2` |
| `x ^ (1/n)` | n 次方根 | `9 ^ (1/2) → 3` |

### 三角函数

| 函数 | 含义 | 示例 |
|------|------|------|
| `sin(x)` | 正弦（弧度） | `sin(pi/6) → 0.5` |
| `cos(x)` | 余弦（弧度） | `cos(0) → 1` |
| `tan(x)` | 正切（弧度） | `tan(pi/4) → 1` |

> 注意：三角函数参数为弧度，而非角度。如需计算角度值，请先转换：`sin(pi * 角度 / 180)`。

### 对数函数

| 函数 | 含义 | 示例 |
|------|------|------|
| `log(x)` | 常用对数（log₁₀） | `log(100) → 2` |
| `ln(x)` | 自然对数（logₑ） | `ln(e) → 1` |

### 其他函数

| 函数 | 含义 | 示例 |
|------|------|------|
| `abs(x)` | 绝对值 | `abs(-5) → 5` |

### 常量

| 名称 | 含义 | 值 |
|------|------|----|
| `pi` | 圆周率 π | 3.14159265... |
| `e` | 自然常数 | 2.71828182... |

## 括号与优先级

优先级从高到低：

1. 括号 `( )`
2. 乘方 `^` / 函数调用
3. 一元正负号 `+` / `-`
4. 乘除 `*` / `/`
5. 加减 `+` / `-`

示例：
- `(1 + 2) * 3 → 9`（括号优先）
- `-3 ^ 2 → -9`（乘方优先于负号，即 -(3²) 而非 (-3)²）
- `(-3) ^ 2 → 9`（括号改变优先级）

## 负数输入

- 直接在数字前加 `-`：`-5.2`
- 负底数需用括号包裹：`(-5.2) ^ 3 → -140.608`
- 负底数的非整数次方在实数域内无定义，会报错：`(-1) ^ 0.5 → 错误`

## 错误处理

| 错误类型 | 触发条件 | 示例 |
|----------|----------|------|
| 除以零 | 分母为零 | `1 / (2 - 2)` |
| 实数域错误 | 负底数的非整数次方 | `(-1) ^ 0.5` |
| sqrt 域错误 | sqrt 参数为负数 | `sqrt(-4)` |
| 语法错误 | 表达式格式不正确 | `2 + * 3` |
| 未识别名称 | 未注册的函数/常量名 | `foo(1)` |

## 交互模式命令

| 命令 | 功能 |
|------|------|
| `help` | 显示帮助信息 |
| `quit` / `exit` / `q` | 退出程序 |
| `Ctrl + C` | 中断当前输入 |
| `Ctrl + D` | 退出程序（EOF） |

## 完整示例

```
>>> 1 + 2 * 3
7
>>> sin(pi / 6)
0.5
>>> pow(25, 1/2) + sqrt(16)
9
>>> (-5.2) ^ 3
-140.608
>>> 1 + 5 * pow(,1/2) / (2 * (-5.2) ^ 3)
0.9822200728265817
>>> pi + e
5.85987448204884
>>> abs(-3.14) * log(1000)
9.42
>>> ln(e ^ 5)
5
```
