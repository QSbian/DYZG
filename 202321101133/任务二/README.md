改好了喵~ 现在直接 `python calculator.py` 就能进入交互模式，持续输入表达式、持续出结果，就像这样：

```
===== 命令行科学计算器 =====
输入 "help" 查看帮助，输入 "quit" 退出喵~

>>> 1+2*3
7
>>> pow(25,1/2)
5
>>> (-5.2)^3
-140.608
>>> 1/(2-2)
[错误] 数学错误：除以零
>>> quit
拜拜喵~
```

输 `quit` / `exit` / `q` 退出，输 `help` 看帮助，Ctrl+C 中断当前输入喵~ 之前那种 `python calculator.py "表达式"` 的单次计算方式也保留了，向后兼容喵~
