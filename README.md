## 任务概览

### 任务一 — Git 操作文档

各成员编写面向三人小团队的 Git 协作指南，覆盖日常命令、分支管理、冲突处理等。

### 任务二 — Python 命令行科学计算器

基于**递归下降解析器**的 CLI 计算器，支持四则运算、乘方 `^`、`pow(m,n)`、三角函数、对数、括号及实数域运算。

### 任务三 — 计算器 GUI / 多端实现

在任务二核心引擎基础上，为计算器添加图形界面或多平台支持：

- **202321101063**: [PySide6 深色主题 GUI 计算器](./202321101063/任务三/) — Catppuccin Mocha 配色，历史记录，键盘快捷键
- **202321101133**: 三端全覆盖
  - 🐍 [Python 增强版 CLI](./202321101133/任务三/calculator.py)
  - 🖥️ [Qt C++ 桌面版](./202321101133/任务三/Desktop/QtCalculator/)
  - 🌐 [Vue 3 网页版](./202321101133/任务三/HTML/calculator-vue/)
  - 📱 [APK 安卓版](./202321101133/任务三/Apk/)
